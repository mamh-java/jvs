## 一、介绍

### 1 概述

文档管理，支持富文本、Excel、脑图等在线编辑预览等功能

### 2 特性

* 富文本
* 脑图
* 流程图
* excel
* 编辑/查看权限 
* 协同人员设置 
* 评论
* 点赞
* 搜索

## Git 提交代码时添加 emoji 图标

```git
git commit -m ':emoji1: :emoji2: :emoji3: 此次提交的内容说明'
```

|:art:|```:art:```|改进结构和代码格式|
|⚡️|```:zap:```|优化性能|

|🐛|```:bug:```|复 BUG|
|🚑|```:ambulance:```|核心代码修复|
|✨|```:sparkles:```|添加新特性|
